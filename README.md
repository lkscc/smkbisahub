# 🏫 SMK BISA Hub - LKS Cloud Computing Jatim 2026

Selamat datang di repositori resmi untuk LKS Cloud Computing Tingkat Provinsi Jawa Timur 2026. 
Repositori ini berisi *source code* untuk aplikasi "SMK BISA Hub".

## ⚠️ Instruksi Arsitek
Sebagai *Cloud Engineer*, tugas Anda bukan membuat aplikasi ini dari awal, melainkan:
1. Men-deploy aplikasi ini ke infrastruktur AWS (EC2 & Amplify) yang memenuhi standar *High Availability*.
2. Menghubungkan aplikasi ke layanan AWS terkelola (RDS, S3, SNS, Secrets Manager).
3. Melengkapi bagian kode yang ditandai dengan `TODO: LKS CHALLENGE`.

## 📂 Struktur Repositori
- `/frontend` : Aplikasi React.js untuk sisi klien (Client-side).
- `/backend`  : Aplikasi Node.js Express untuk sisi server (API).
- `/database` : Skrip SQL inisialisasi tabel RDS.
- `/machine-learning` : Template Jupyter Notebook untuk Amazon SageMaker.
- `/iac` : Template kerangka kerja AWS CloudFormation.

*Semoga berhasil. Telitilah dalam membaca error log!*
